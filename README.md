# 🌿 Fornel Consultoria

Site desenvolvido para a Fornel Consultoria, com o objetivo de apresentar serviços, fortalecer a presença digital e facilitar o contato com clientes.

## 🌐 Acesse o projeto
https://site-fornel-consultoria.vercel.app

## 📌 Sobre o projeto
O projeto consiste em um site moderno e responsivo criado para representar a identidade da empresa Fornel Consultoria na web, destacando seus serviços, proposta de valor e canais de comunicação.

## 🎯 Objetivos
- Apresentar os serviços da consultoria de forma clara
- Gerar credibilidade e presença digital
- Facilitar o contato com clientes
- Melhorar a experiência do usuário

## 🛠️ Tecnologias utilizadas
- JavaScript
- HTML5 & CSS3
- Vercel (deploy)

## 💻 Funcionalidades
- Layout responsivo
- Navegação intuitiva
- Seção de serviços
- Página institucional
- Contato direto 

## 🎨 Design
O design foi desenvolvido com foco em:
- Clareza na comunicação
- Experiência do usuário (UX)
- Identidade visual profissional
